# EchoX
EchoX Is A Programming Language That Only Has 1 Command, Let me show you how:

## Cloning the repository
```bash
git clone https://github.com/joelclaude0817-beep/EchoX/
```

## Changing Directories
```bash
cd EchoX
```

## Compiling And Running The Programming Language
```bash
gcc -o echox echox.c
./echox
```

## Testing The Programming Language

```echox
echo Hello, World!
```
